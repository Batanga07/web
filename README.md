# Entice.Web

Serves a web frontend for entice. Use this to access the worlds.

Needs:

- Erlang version: 17.1
- Elixir version: 1.0.2

To start:

1. Install dependencies with `mix deps.get`
2. Start server with `mix phoenix.start`
